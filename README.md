# Análise de Vendas com SQL

Projeto de análise de dados desenvolvido com SQL, focado na construção de consultas analíticas para acompanhamento de indicadores de desempenho comercial e identificação dos principais drivers de resultado.

O projeto explora dados de vendas com o objetivo de transformar informações brutas em insights estratégicos para apoio à tomada de decisão.

---

# Objetivo do Projeto

Desenvolver análises orientadas a negócio utilizando SQL, aplicando consultas analíticas para explorar performance comercial, comportamento de vendas e métricas estratégicas.

As análises realizadas buscam responder perguntas como:

- Quais categorias geram maior faturamento?
- Quais produtos possuem maior volume de vendas?
- Como as vendas evoluem ao longo do tempo?
- Quais clientes possuem maior participação na receita?
- Quais fatores impactam os resultados do mês?

---

# Tecnologias Utilizadas
